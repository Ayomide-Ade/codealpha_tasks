# CodeAlpha_Phishing_Awareness

An interactive Phishing Awareness Training module developed as part of the **CodeAlpha Cyber Security Internship**.

---

## Objective

To educate users on how to identify, avoid, and respond to phishing attacks through structured content and an interactive knowledge check.

---

## Features

- **Educational Content** — Defines phishing and explains the psychological tactics attackers use, including urgency and social engineering.
- **Threat Identification** — Highlights common red flags in fake emails and websites such as mismatched domains, generic greetings, and suspicious links.
- **Best Practices** — Outlines actionable prevention steps including Multi-Factor Authentication (MFA), source verification, and incident reporting.
- **Interactive Quiz** — A JavaScript-based knowledge check that tests the user's ability to respond correctly to a simulated CEO fraud attempt, with tailored feedback for each answer choice.
